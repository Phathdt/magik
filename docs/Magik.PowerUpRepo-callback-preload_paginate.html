    preload_paginate({entries :: list(), paginator :: map()}, preloads :: keyword()) ::
  {list(), map()}

    Preload entries return from Repo.paginate

    Example

    {entries, paginator} =
  from(p in Product, where: p.quantity > 10 )
  |> Repo.paginate(%{page: 2, size: 5})
  |> Repo.preload([:brand, :category])
